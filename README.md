# CineEnglish

Streamlit app for learning English through movies and series.

## Run

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run main.py
```

## Env vars (optional)

- `GROQ_API_KEY`
- `TMDB_API_KEY`
- `OPENSUBTITLES_API_KEY`
- `EMAIL_FROM`, `EMAIL_TO`
